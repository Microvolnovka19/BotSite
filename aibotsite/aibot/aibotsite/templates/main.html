API

1. Загрузка файла на сервер к боту с айди 4
curl -X 'PUT' '127.0.0.1:8000/api/v1/bot/4/upload/' -F 'file=@img.png'

2. Удаление бота с айди 4
curl -X 'DELETE' '127.0.0.1:8000/api/v1/bot/4/delete/'

3. Создание нового бота БЕЗ ФАЙЛА. Файл загружается отдельной командой(см первый пункт)
curl -X 'POST' '127.0.0.1:8000/api/v1/bot/new/' -H 'Content-Type: application/json' -d'{"name":"CURL","author":"AVTOR","short_desc":"OPICANIE"}'

4. Получить список ботов
curl -X 'GET' '127.0.0.1:8000/api/v1/bot/'

5. Получить инфу о боте 1
curl -X 'GET' '127.0.0.1:8000/api/v1/bot/1/'

6. Получить файл бота с айди 1
curl -X 'GET' '127.0.0.1:8000/api/v1/bot/1/file/'

7. Обновить инфу о боте 5. БЕЗ ФАЙЛА. Файл меняется через первый пункт
curl -X 'PUT' '127.0.0.1:8000/api/v1/bot/5/edit/' -H 'Content-Type: application/json' -d'{"name":"CURL","author":"AVTOR","short_desc":"OPICANIE"}'
